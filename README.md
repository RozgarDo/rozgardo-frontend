# RozgarDo Frontend

React-based user interface for the RozgarDo job portal.

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- React Router DOM

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Build

```bash
npm run build
```

## Features

- User authentication (Employee, Employer, Admin roles)
- Job listings and search
- Application tracking
- Profile management
